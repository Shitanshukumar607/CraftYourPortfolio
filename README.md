# Resume Craft

Turn your resume into a portfolio in seconds. Showcase your professional journey with a beautifully generated portfolio website, because your career deserves more than just a PDF.

![App Mockup](./screenshots/page.png)

## Features

- **Instant Conversion**: Upload your resume and get a portfolio site generated automatically.
- **Modern Design**: Clean, responsive UI built with the latest web technologies.
- **Drag & Drop**: Easy file upload interface.

## Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/Shitanshukumar607/CraftYourPortfolio.git
   cd resume-craft
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the development server**

   ```bash
   npm run dev
   ```

4. **Open the app**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.
